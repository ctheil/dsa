# Binary Tree

## BT Rules

  1. No children
  2. One child
  3. two child

## Types

- Full BST
  - All children terminate with 2 leaf nodes
- Partial complete BST
  - right-most child is missing rightmost leaf
- Complete bst
  - right-most child is a leaf

## Syntax

```mathematica
        |xx| <- Parent
   |xx| <-Uncle |Node|
|xx| |xx|     |xx| |xx| <- Children (siblings to one another)
```
